== Building image

[source,bash]
----
$ docker build -t kalemena/tensorflow:latest .
----

== Running image

[source,bash]
----
$ docker run -p 8888:8888 -v "$PWD/notebooks:/opt/notebooks" -d kalemena/tensorflow:latest
----

== Open Jupyter

Open http://localhost:8888/ (password is 'root')
